# Kontakt AI - Müştəri İtki Riski (Churn Prediction)

Bu sistem, müştərilərin alış tezliyi və rəy ballarını analiz edərək, onların rəqibə keçmə riskini proqnozlaşdıran bir süni intellekt modelidir.

## Biznes Dəyəri
- Müştəri itkisinin (churn) qarşısını almaq üçün erkən xəbərdarlıq mexanizmi.
- Satış şöbəsi üçün prioritetli müştəri siyahısının avtomatik yaradılması.

## Texnologiya
- Python (Data Analysis & Logic)
